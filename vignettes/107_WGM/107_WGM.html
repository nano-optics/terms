Objective

This example illustrates the calculation of whispering gallery modes in a dimer of large silicon spheres.

This simulation uses the following input file for far-field cross-sections

ModeAndScheme 2 3
MultipoleCutoff 15
Wavelength 700 900 50
Verbosity 1
Medium 1.0

Incidence 0 1.570796 0 2
OutputFormat HDF5 xsec

Scatterers 2
Si 0.0  -1050.0 0.0 1000.0
Si 0.0  1050 0.0 1000.0

and for near-field maps

ModeAndScheme 1 0
MultipoleCutoff 20
Wavelength 600 900 2
Verbosity 1
Medium 1.0

Incidence 0 1.570796 0 2
OutputFormat HDF5 map
SpacePoints -1500 1500 200 -2500 2500 400 0 0 0 0 0 0 
MapQuantity 2 E
